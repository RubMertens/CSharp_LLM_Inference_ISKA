SwiGLU Activation

The non-linear "thinking" step between attention layers

Why do we need activation functions?

Without them, stacking layers is pointless — N linear layers = 1 linear layer.

Matrix × Matrix × Matrix = just another Matrix. Activation functions break this linearity and let networks learn complex patterns.

SwiGLU(x) = Swish(xW₁) ⊙ (xW₂)

Swish(x) = x × sigmoid(x) — a smooth version of ReLU

⊙ = element-wise multiplication (the "gating" mechanism)

The gate learns which parts of the signal to keep and which to suppress
