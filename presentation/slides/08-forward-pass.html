The Forward Pass

The model looks at all previous tokens and predicts what comes next.

Simplified
Input:  "The cat sat on the"
                    ↓
          [ Transformer Layers × N ]
                    ↓
Output: { "mat": 0.12, "floor": 0.08,
          "roof": 0.04, "dog": 0.001, ... }

What actually happens

  • Tokens → embeddings (vectors)
  • Attention: each token "looks at" all others
  • Feed-forward layers refine predictions
  • Repeat for each transformer layer

Key insight

The output is a probability distribution over the entire vocabulary — not a single answer. The sampler decides which token to pick.
